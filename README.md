# Portfolio Personal

## Descripción

Sitio web personal que presenta mi experiencia profesional, proyectos y habilidades técnicas en Data Science y Machine Learning.

## Contenido

- **Información profesional**: Perfil y experiencia laboral
- **Proyectos**: Muestra de trabajos realizados
- **Habilidades técnicas**: Tecnologías y herramientas con las que trabajo
- **Formación académica**: Estudios y certificaciones
- **Multiidioma**: Disponible en español e inglés

## Deploy

El sitio está desplegado en Vercel.
